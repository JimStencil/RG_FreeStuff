Jigsaw Puzzle Game Template (Corona SDK version)

This template is graphics 1.0 and graphics 2.0 (compatibility mode only) ready.

This popular template shows you how to implement two kinds of complete Jigsaw Puzzle Games, using either:
- Rectangular Pieces, or
- Jigsaw shaped pieces.

Contains:
- 4 Pure versions,
- 4 SSK versions,
- 1 Hybrid version, 
- 1 Guide to making rectangular pieces,
- 1 Guide to make jigsaw shaped pieces, and 
- 1 General reference guide to the kit and SSK.
